# BabylonJSTest

This is the public repository for Babylon.js contents.

## Publish to Github Reposotory

Just follow these commands.
```
git init
git add .
git commit -m "The Nth commit on date."
git remote rm origin
git branch -M main
git remote add origin git@github.com:robin-artemstein/babylonjs-test.git
git push -u -f origin main
```
